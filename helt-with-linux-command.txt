Для развертывания проекта лкально, нужно:
1) В vscode открыть нужную папку 
2) Выполнить команду git init (создает скрытую папку .git и делает текущую директорию корневой)
3) Выполнить команду git clone (ссылка на репозиторий в GitHub)
4) Дождаться окончания скачивания проекта
5) Выполнить команду cd (имя проекта) (для перехода в заданную папку)
6) Выполнить команду git init (создает скрытую папку .git и делает текущую директорию корневой)
7) Далее выполняем команду git status, чтобы убедиться, что всё сделано правильно
8) Начинаем отслеживать изменения в файлах git add . (все файлы добавляются в индексную область)
Готово!
При работе с проектом, когда потребуется внести изменения в репозиторий:
0.5) При необходимости смотрим что изменилось - git status
1) Добавляем изменения в файлах git add . (все файлы добавляются в индексную область)
2) Создаем коммит с описанием внесенных изменений
1.5) Новый git status говорит о том, что локальный репозиторий новее на сколько-то коммитов удаленного репозитория
2) Если изменений больше вносить не требуется выполняем git push -u origin main (отправляет изменения на удаленный репозиторий)
